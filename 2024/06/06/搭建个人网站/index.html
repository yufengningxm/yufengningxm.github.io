配置博客的环境

确保缩进和语法正确。完整示例如下：

node -v
npm -v

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

hexo g #生成静态文件
hexo s本地预览

deploy:
type: git
repo:
branch: gh-pages

npm install hexo-deployer-git –save

server:
port: 8888
compress: true
header: true

hexo s -p 5000

每次运行都要换，有点麻烦
