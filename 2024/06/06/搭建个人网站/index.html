配置博客的环境

Node.js

Node.js是JavaScript语言运行环境，用于开发服务器端和网络应用程序。
npm (node package manager)是 JavaScript 世界的包管理工具,并且是 Node.js 平台的默认包管理工具。

安装

官网下载：nodejs.org
方法1: 复制代码到终端

方法2: 直接下载安装包，一直点下一步

1.选择对应的版本号。其中，Current是最新开发版本，包含最新的功能、改进和实验性特性，但不太稳定。而LTS版（Long-Term Support，长期支持版）提供稳定且可靠的版本，适合生产环境中的应用程序，建议选择LTS版。
2.选择自己的电脑系统（Windows、macOS、Linux）
安装成功后查看node和npm版本

Hexo

Hexo是基于Node.js的静态博客框架Hexo。

安装

终端切换到超级用户模式
sudo su
(输入密码)

创建博客目录并初始化

blog文件夹下生成

终端显示
sh-3.2# npm install hexo-cli -g
changed 53 packages in 7s
14 packages are looking for funding
run npm fund for details
sh-3.2# hexo init blog
INFO Cloning hexo-starter https://github.com/hexojs/hexo-starter.git
INFO Install dependencies
INFO Start blogging with Hexo!
sh-3.2# cd blog
sh-3.2# npm install
hexo server

23 packages are looking for funding
run npm fund for details

found 0 vulnerabilities
sh-3.2# hexo server
INFO Validating config
INFO Start processing
INFO Hexo is running at http://localhost:4000/ . Press Ctrl+C to stop.

在网页输入http://localhost:4000/ 进入到博客初始页面

control+c退出界面

到这里，算成功创建了个人网站，接下来是网站部署到GitHub，以及打造专属域名的教程。

GitHub部署

配置SSH keys，部署到GitHub Pages

设置_config.yml中的deploy选项

安装部署插件

部署到GitHub Pages
hexo deploy

主题更换

下载主题到themes文件夹：
git clone themes/
修改_config.yml中的theme值为新主题名。
发表文章：
hexo new “My New Post”
编辑器会自动打开，您可以根据提示填写相关信息。
更新文章并重新生成和部署：
hexo generate
hexo deploy
